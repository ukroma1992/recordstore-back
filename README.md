# Ruby on Rails API

Простое api написаное на Ruby on Rails. Для регистрации было использовано jwt(JSON Web Token)

#### Здесь используеться:

- `Rails 5.2.2`
- `Ruby 2.5.3`
- Gem `bcrypt 3.1.7`
- Gem `rack-cors`
- Gem [`redis 4.0`](https://github.com/redis/redis-rb)
- Gem [`jwt-sessions`](https://github.com/tuwukee/jwt_sessions)
